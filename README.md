# PulseIQ AI

PulseIQ AI is an AI-powered clinical document intelligence platform built on AWS.

## Features
- Upload clinical PDFs
- Automated OCR processing using Amazon Textract
- Bedrock Knowledge Base integration
- Multi-agent AI system using Amazon Bedrock Agents
- Semantic retrieval using OpenSearch Serverless
- Real-time clinical querying through Streamlit UI

## AWS Services Used
- Amazon S3
- AWS Lambda
- AWS Step Functions
- Amazon Textract
- Amazon Bedrock
- OpenSearch Serverless
- DynamoDB
- API Gateway
- EC2

## Architecture
Event-driven pipeline for ingestion, processing, retrieval, and multi-agent reasoning.

## Tech Stack
Python, Streamlit, AWS, Bedrock, DynamoDB
